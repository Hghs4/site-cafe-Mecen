
# Site internet M1 Mecen : Le Café d'Antan

Projet M1 MECEN 

Ce projet a été réalisé dans le cadre du cours de Développement Web (HTML/CSS) du Master MECEN.
L’objectif était de créer un site web personnalisé, en utilisant seulement du css et du html tout en respectant les contraintes données : HTML5, CSS3 pur, utilisation de Flexbox/Grid, Google Fonts, Font Awesome...

---

## Objectifs du projet

Concevoir un site web vitrine simple autour d’un thème libre.
J’ai choisi de créer le site d’un café vintage fictif, avec plusieurs pages et une navigation classique.

---

## Structure du site

Le site a **6 pages** :

- Accueil

- Histoire

- Spécialités

- Galerie 

- Équipe

- Contact
